qwx
===

微信 Qt 前端，使用[网页微信协议] (https://github.com/xiangzhai/qwx/blob/master/doc/protocol.md) 请不要用于商业用途！图片、协议版权归[腾讯] (http://weixin.qq.com/) 所有。


## 依赖

Qt5(>=5.2)的qml、quick、network、xml 


## 编译、运行

```
qmake QWX_DEBUG=ON
make
./qwx
```
